Projeto Ágape

O Projeto Ágape foi desenvolvido visando facilitar e automatizar o processo de doação para ONG’s, por meio da qual você, doador, poderá visualizar o perfil de cada organização onde prefira doar.

Se você é responsável por uma organização, também pode entrar nesse projeto conosco! Crie um perfil para a organização pela qual é responsável, onde o doador poderá visualizar o seu projeto e entrar em contato para realizar uma doação.
